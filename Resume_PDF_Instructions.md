# How to Convert Resume to PDF and Upload to LinkedIn

## Step 1: Convert Resume to PDF

### Option A: Using Mac Preview (Easiest)

1. **Open the resume file** (`Keith_Drewnoski_Resume_General.txt`) in TextEdit or any text editor
2. **Select All** (Cmd+A) and **Copy** (Cmd+C)
3. **Open Pages** (or Microsoft Word if you have it)
4. **Paste** the text
5. **Format it nicely:**
   - Make your name larger and bold
   - Add spacing between sections
   - Make section headers bold
   - Adjust margins if needed
6. **Export as PDF:**
   - File → Export To → PDF
   - Choose quality (High is fine)
   - Click Export
   - Save as: `Keith_Drewnoski_Resume.pdf`

### Option B: Using Google Docs (Free, Easy)

1. **Go to** https://docs.google.com
2. **Create a new document**
3. **Paste** the resume text
4. **Format it:**
   - Make name larger and bold
   - Add spacing between sections
   - Make section headers bold
5. **Download as PDF:**
   - File → Download → PDF Document (.pdf)
   - Save as: `Keith_Drewnoski_Resume.pdf`

### Option C: Using Online Converter

1. **Copy** the resume text
2. **Go to** https://www.ilovepdf.com/text-to-pdf (or similar)
3. **Paste** text
4. **Convert to PDF**
5. **Download** the PDF

---

## Step 2: Upload to LinkedIn

### Method 1: Featured Section (Recommended)

1. **Go to LinkedIn** → Your Profile
2. **Scroll down** to the "Featured" section (below your About section)
3. **Click the "+" button** or "Add featured"
4. **Select "Media"**
5. **Upload** your PDF resume (`Keith_Drewnoski_Resume.pdf`)
6. **Add a title:** "Keith Drewnoski - Resume 2025" or "Resume - QA & Software Development"
7. **Click Save**

### Method 2: Contact Info Section

1. **Go to LinkedIn** → Your Profile
2. **Click "Contact info"** (under your name/headline)
3. **Scroll down** to find resume section (if available)
4. **Upload** your PDF resume

### Method 3: About Section (Less Common)

Some profiles allow attaching documents in the About section, but Featured is better.

---

## Step 3: Verify It's Visible

1. **View your profile** as others would see it
2. **Check Featured section** - your resume should appear there
3. **Test the download** - click on it to make sure it opens correctly

---

## Tips

✅ **File Name:** Use a clear name like `Keith_Drewnoski_Resume_2025.pdf`  
✅ **File Size:** Keep it under 5MB (PDFs are usually small)  
✅ **Formatting:** Make sure it looks good when downloaded  
✅ **Update Regularly:** Update it when you gain new experience

---

## Troubleshooting

**If PDF won't upload:**

- Check file size (should be under 5MB)
- Make sure it's a PDF file (.pdf extension)
- Try a different browser
- Clear browser cache

**If formatting looks off:**

- Make sure you formatted it in Pages/Word/Google Docs before exporting
- Check margins and spacing
- Test download to see how it looks
